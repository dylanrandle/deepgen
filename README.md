# deepgen

**Deep convolutional neural networks for generative modeling (VAE, GANs, etc.)**

## Requirements
Tested with:
- `python==3.7.3`
- `torch==1.3.0`
- `torchvision===0.4.1`
- `numpy==1.16.4`
- `matplotlib==3.1.0`

## Simple usage
- `python cnn/train_vae.py` (add `-h` for available arguments)
